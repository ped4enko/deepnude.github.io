Create Bitmap Movies

 Top  Previous  Next

Description

Specifies the Mission Plan to use when generating movies.

 

Forms

•--create-bitmap-movies

•-cbm

 

Usage

 

FreeFlyer.exe --create-bitmap-movies <PATH TO FILE>

 

Example

 Specify "default.MissionPlan" as the Mission Plan to use to generate a movie.

 

FreeFlyer.exe -cbm default.MissionPlan
