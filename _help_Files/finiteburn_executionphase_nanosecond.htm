FiniteBurn.ExecutionPhase Property

 Top 

Description

Reveals which portion of the event sequence is currently executing.

 

Timing Precision Mode

This page describes functionality in nanosecond timing precision mode.

 

Attributes

Type:

integer

Access:

read

Valid Values:

Value

Label

0

Inactive

1

Propagating to start time.

2

Propagating through burn.

 

Syntax

myVariable1 = myFiniteBurn1.ExecutionPhase;
